# Inizio Test Project

Simple web application built with FastAPI.

## Features

- Search query input
- Fetches Google organic results (via SerpAPI)
- Returns structured JSON (title, link, snippet)
- Unit test included (pytest)

## Tech Stack

- FastAPI
- Python
- JavaScript (fetch API)

## Run locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
